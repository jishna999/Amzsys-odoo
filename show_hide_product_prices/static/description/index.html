PRODUCTION CREATING VIA BILL OF MATERIAL

By Accurate IT Solutions

This module helps to create production order via bill of material

Description with screenshots

The following screenshot demonstrate how to add production order through wizard button on Bill of Material

In Manufacture module select the option Product and choose Bill of Material From dropdown .

Click New create a new Bill of Material .

Choose Product and add components and their quantity in Components tab and save the Bill of Material. Click on the wizard button Create Manufacture Order

Opens Wizard of Creating Manufacture Order Choose Responsible user,Quantity and scheduled date. Click the button Create manufacture order in the wizard

This Created new Two Manufacture of that Bill of Material in by different responsible user . Then open newly added Manufacture Order

Here in this manufacture order responsible user,Quantity,schedule date and components are updated according to the Bill of Materials
